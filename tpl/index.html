룰렛 모듈 설정

룰렛을 1회 돌릴 때 차감할 포인트를 입력하세요. (0이면 무료)

룰렛이 멈출 때까지 걸리는 시간입니다. (기본 3초)

JSON 형식으로 아이템을 설정합니다.

[
    {
        "text": "1000",             // 룰렛에 표시될 큰 글씨
        "subText": "포인트",        // 룰렛에 표시될 작은 글씨
        "color": "#FFFFFF",         // 배경색
        "textColor": "#6A4DFF",     // 글자색
        "weight": 0.5,              // 당첨 확률 (가중치) - 높을수록 당첨 잘 됨
        "is_point": true,           // 포인트 지급 여부 (true/false)
        "point_reward": 1000        // 지급할 포인트 양 (is_point가 true일 때)
    },
    {
        "text": "꽝",
        "subText": "다음에 다시",
        "color": "#e2e8f0",
        "textColor": "#64748b",
        "weight": 50,
        "is_point": false,
        "point_reward": 0
    }
]
